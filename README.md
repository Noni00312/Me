# 🚀 Task Manager / Goal Tracker / Journey Tracker

## 📌 Overview
**Task Manager / Goal Tracker / Journey Tracker** is a productivity-focused application designed to help users manage tasks, track goals, and document their learning journey in one unified system.

This project combines **task management**, **goal tracking**, and **learning documentation** to give users better visibility into their progress, productivity, and personal growth.

In the future, the system may integrate AI capabilities to automatically generate blog-style summaries based on learning activities.

---

## 🎯 Key Features

### 📂 Project Management
- ➕ Add tasks  
- ✏️ Update tasks  
- ❌ Delete tasks  
- ⏱️ Track task completion time  
- ▶️ Start tasks  
- ⏸️ Pause tasks  

---

### 🎯 Goal Tracker
- 📊 Track progress across multiple projects  
- 📈 Display completion percentage per project  
- 🔄 Monitor active and completed projects  

---

### 📘 Learning Tracker
- 🧠 Add learning topics  
- 📝 Add notes or comments  
- 🔗 Attach useful links  
- ❓ Add questions  
- ✏️ Update/Delete questions  
- 💡 Add answers  
- ✏️ Update/Delete answers  

---

## 💡 Purpose
This project was built to:
- Improve productivity and organization  
- Track both **short-term tasks** and **long-term goals**  
- Create a structured way to document learning  
- Serve as a foundation for future AI-powered features
